# Vast-Launcher
A HTML game launcher Mainly Eaglercraft focused.

It has a File Manger style. 

You can add any type of HTML file it is not restricted to Eagler craft.

It is built to run locally on your computer not online, if ran online you may run into bugs, DO NOT ASK FOR SUPPORT ON THAT IF DOING SO.

One last thing, I am not going to respond to every email and if your question is obvious and I think you did no effort to fix it yourself (If it is possible to do so) then I will ignore it. Feature requests are in person or via this github only don't email me feature requests I won't asnwer at all.

When Emailing me Put the title as "Vast Bug" it makes it easier for me and you. Also please mention your web browser, and OS I will work on it as soon as I can.
